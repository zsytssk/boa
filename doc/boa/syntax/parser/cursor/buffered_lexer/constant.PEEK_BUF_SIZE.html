[−][src]Constant boa::syntax::parser::cursor::buffered_lexer::PEEK_BUF_SIZE

const PEEK_BUF_SIZE: usize = (MAX_PEEK_SKIP + 1) * 2; // 6usize

The fixed size of the buffer used for storing values that are peeked ahead.

The size is calculated for a worst case scenario, where we want to peek MAX_PEEK_SKIP tokens skipping line terminators:

[\n, B, \n, C, \n, D, \n, E, \n, F]
  0  0   1  1   2  2   3  3   4  4
